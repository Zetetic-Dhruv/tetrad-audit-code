================================================================================
JUDGE PROTOCOL — per-run record
================================================================================

This file is a TEMPLATE. Each substrate run should drop a copy of this file
named `judge_protocol_<substrate>_<YYYYMMDD>_<HHMM>.txt` alongside its outputs/
directory, with the fields filled in. If the running agent forgets to drop
this file, the same information is recoverable post-hoc by querying outputs/
(e.g., counting escalations, distinct evaluator strings, timestamp ranges).

The protocol query script (to be written) reads outputs/*.json and emits
per-substrate aggregates: this template lets the human auditor verify the
auto-generated aggregates against the agent's locally-recorded protocol.
